SmartCalc

О проекте

В данном проекте реализована программа SmartCalc на языке программирования СИ с использованием структурного подхода. Программа позволяет вычислять помимо базовых арифметических операций вычисления арифметических выражений с учетом приоритетов. Помимо этого программа доаолнена некоторыми арифметическими функциями и имеет возможность выполнять построение графиков. Дополнительно был реализован кредитный калькулятор.

Арифметические операторы

  • ()
  • +
  • -
  • *
  • /
  • ^
  • mod
  • Унарный плюс
  • Унарный минус

Функции

  • sin(x)
  • cos(x)
  • tan(x)
  • acos(x)
  • asin(x)
  • atan(x)
  • sqrt(x)
  • ln(x)
  • log(x)

Установка

Для использования калькулятора нужно использовать команду make install.

Удаление

Для удаления калькулятора нужно использовать команду make uninstall.
